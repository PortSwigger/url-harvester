This Burp Suite Extension, URL Harvester, captures in-scope URLs from Burps Proxy and Target tools. It provides a graphical user interface (GUI) to view, manage, and export them. URLs can be stored and navigated for an easier analysis and a further detailed inspection.

Features

URL Harvester captures in-scope URLs and allows users to:

  • View the captured URLs on the interface.
  • Manage the list of URLs for easier control and functionality.
  • Export URLs for use in other tools or for report generation.

Usage

To use URL Harvester, navigate to the URL Harvester tab within Burps user interface. URLs captured by the Proxy and Target tools will be automatically populated and organised within this tab. Manage the URLs as necessary or export them for further use.
